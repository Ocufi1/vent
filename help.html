The Vent

This server's general purpose is to give people a simple but unique server for mainly chatting, but also gaming too. We host lots of giveaways and events. Then on top of that we have a huge variety of emojis to use in the server and of course outside of the server if you have nitro. Everyday we're working hard to make this server the best it could possibly be. There's always knew things being developed within the server, so you don't just feel like it's been the same since you got here. Although it does take time as the server's content is only developed by the owner. Just some things we do that we personally think puts above a lot of other servers is talking one on one with people we mute, reviewing each ticket that has ever been created, building a website instead of signing up for a patreon, not ignoring new people so they feel welcomed, we'll respond to your dm if your not asking for things, also allowing you to moderate your own voice channels because we know how hard it is to report someone that correlates with a voice channel, and the list goes on. We would really appreciate it if you joined the server and stayed as long as possible. Thank you for reading this and we hope you have a great day!
